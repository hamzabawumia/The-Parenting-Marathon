# 👶 The Parenting Marathon 🏃‍♂️📘

*A year-by-year guide to raising a child from Age 1 to 20.*

---

## 🌱 About This Project

**Parenting Marathon** is a long-term guide that breaks parenting into stages—one year at a time.

Each page focuses on:

* Development milestones
* Emotional needs
* Practical parenting tips
* Life Skills Couching
* Home Schooling
